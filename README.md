## Антивирус на Python

## Загрузка данных

Проект начинается с загрузки данных из CSV-файлов с помощью библиотеки pandas. Данные разделены на обучающий (train) и тестовый (test) наборы. Они содержат ключевую информацию о пассажирах "Титаника", которая служит основой для всего анализа.

## Этот проект демонстрирует, как работает простой антивирусный движок, используя:

Проводя Разведочный анализ данных (EDA) , я погружаюсь в глубокое изучение набора данных. С помощью различных методов визуализации я:

* Сигнатурное сканирование
* Хеши SHA256 для определения вредоносных файлов
* Правила YARA
* Анализ PE-файлов (Windows)
* Сканирование процессов
* Многозадачность
* Карантин для заражённых файлов

## Установка

Клонируйте репозиторий:

```git clone https://github.com/yourname/python-antivirus

cd python-antivirus```

Установите зависимости:

```pip install -r requirements.txt```

## Зависимости

* psutil
* yara-python
* pefile

## Как запустить?

Напишите в терминале комманду:

```python src/scanner.py```

Затем введите папку для сканирования:

```Directory to scan: C:\\Downloads```

Пример вывода:

```\[!] Threat detected: Suspicious\_PowerShell

File: C:\\Downloads\\malware.exe```
